# 🔐 PRODIGY_CS_02 - Image Encryption Tool

## 📌 Task-02: Pixel Manipulation for Image Encryption

This project is part of my internship at Prodigy Infotech, where I developed a simple image encryption and decryption tool using pixel manipulation techniques.

## 🚀 Features
- Encrypt images using XOR operation
- Decrypt images back to original
- Pixel scrambling for extra security

## 🛠️ Technologies Used
- Python
- OpenCV
- NumPy

## ▶️ Installation

```bash
pip install opencv-python numpy
```

## ▶️ Run

```bash
python image_encryptor.py
```

## 🔑 Concept

Encrypted Pixel = Original Pixel XOR Key

Decrypted Pixel = Encrypted Pixel XOR Key
